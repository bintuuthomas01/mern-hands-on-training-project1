# Mern Stack hands on training project1:Simple Product Management System

A lightweight, full-stack web application built using the MERN stack (MongoDB, Express.js, React, Node.js). This project was developed as a hands-on learning application during college training by zoeverse tech to understand core CRUD operations and full-stack integration.

## 🚀 Features

*   **Create:** Add new products with details (Name, Price, Description, etc.).
*   **Read:** View a dynamic list of all available products fetched directly from the database.
*   **Update:** Edit existing product details seamlessly.
*   **Delete:** Remove products from the inventory in real-time.

---

## 🛠️ Tech Stack

### Frontend
*   **React.js** - Component-based user interface.
*   **Axios** - For handling API requests to the backend.

### Backend
*   **Node.js** - JavaScript runtime environment.
*   **Express.js** - Minimalist web framework for building the RESTful API.

### Database
*   **MongoDB** - NoSQL database for flexible data storage.
*   **Mongoose** - ODM (Object Data Modeling) library for MongoDB and Node.js.

---

## 📌 Project Status & Disclaimer

> **Note:** This repository is a simple, hands-on learning project developed during college training to understand the fundamentals of full-stack MERN development. It is **not** intended to be a fully featured production application. The primary focus was on implementing and understanding basic CRUD operations, database integration, and API routing.
